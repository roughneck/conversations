= Conversations

This project rocks big time and uses MIT-LICENSE.

= Installation

* Add
  gem "conversations"
  to your Gemfile.

* Run
  bundle install
  .

* Add
  //= require conversations
  to your app/assets/javascripts/application.js file.

* Add
  *= require conversations
  to your app/assets/stylesheets/application.css file.

* Run
  rails g conversations
  .

* Run
  rake db:migrate
  .
